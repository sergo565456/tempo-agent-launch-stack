# Agent Launch Intel Stack

Public source bundle for Agent Launch Intel API and its Tempo outbound signer.

Projects:

- `tempo-agent-intel-api` - public paid Tempo MPP API.
- `tempo-outbound-signer` - policy-gated remote signer boundary.

Local-only runtime material is intentionally excluded: `.secrets/`, `.data/`, `.vercel/`, `.agents/`, `.tmp/`, `node_modules/`, generated output, and test artifacts.

See each project README for setup and security notes.
